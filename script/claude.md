## 커밋 자동으로 짜주기

지금 변경사항에 있는 모든 파일들에 대한 git commit 을 진행해줘, 모듈별로 그룹하여 커밋 및 커밋메세지를 적절히 작성하여 커밋해. 단 커밋 메세지 양식은 add: , mod: , fix: , chore: 등으로 시작하도록.
-> Please commit all files in the current changes to Git, and group them by module for a clean commit and commit message. The commit message format should start with "add: " for new features, "mod: " for modifications, "fix: " for bug fixes, and "chore: " for miscellaneous tasks.
